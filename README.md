Linux Mysql & MariaDB Lib
===================================

### 의존성 라이브러리 설치  ###
libmysqlcppconn-dev_1.1.9-1_amd64.deb 를 설치해야 한다.
(Mysql & Mariadb library)

Command:

    $ sudo apt-get update
    $ sudo apt-get install libmysqlcppconn-dev
